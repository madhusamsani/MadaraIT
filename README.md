# MadaraIT site (Eleventy)

This branch contains an Eleventy scaffold for a refreshed corporate website for MadaraIT.

How to run locally:

1. Install dependencies: npm install
2. Start dev server: npm start
3. Build for production: npm run build

Notes:
- The existing css/ and assets/ folders are passthrough-copied.
- Place generated AI images in assets/ and reference them from templates.
